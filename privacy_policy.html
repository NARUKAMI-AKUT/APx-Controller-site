1. はじめに

APx Controller（以下「本アプリ」）は、NARUKAMI AKUT が提供する、Audio Precision社製測定器 APx500シリーズ（APx515 等）を遠隔操作するための Windows デスクトップアプリです。本ポリシーは、本アプリがどのような情報を扱うかを説明します。

2. 収集する情報

情報の種類用途外部送信
測定器の設定値・測定データ APx500 アプリケーションの遠隔操作・表示 なし（PC内のみ）
レポート保存先フォルダのパス シーケンス実行レポートの保存先指定 なし（PC内のみ）
利用状況データ（匿名ID・起動日時・アプリバージョン・OSバージョン・エラー種別のみ） アプリの利用状況把握・品質改善 あり（Google Analytics、匿名集計のみ）

本アプリはアカウント登録・ログイン機能を持たず、氏名・メールアドレス等の個人識別情報は一切収集しません。設定や測定データはアプリの動作中のみメモリ上で保持され、ファイルへの保存やサーバーへの送信は行いません。

利用状況データの送信に使われる匿名ID（ランダムなUUID）は、初回起動時に自動生成され、PC内の設定ファイルに保存されます。個人を特定する情報は含まれず、測定値・測定器の設定値・ファイルパス・ユーザー名は一切送信されません。送信される情報は「起動した」「どのバージョンか」「どの種類のエラーが起きたか（エラーメッセージ本文や発生箇所は含まない）」のみです。

なお、この送信には通信時のIPアドレスが利用され、Google Analytics側でおおよその国・地域に変換されます（どの国・地域で使われているかの集計に利用）。IPアドレス自体は保存されず、変換後は破棄されます。

3. 第三者サービス

本アプリは、アプリの起動回数や品質改善のため、Google Analytics（GA4 Measurement Protocol）を利用して匿名の利用状況データを送信します。送信内容は本ポリシー第2項の通りで、広告目的では利用されません。クラッシュレポート専用のサービスは利用していません。

この送信は、起動画面の「使用状況送信」チェックボックスでいつでも無効化できます。無効化した場合、本項の送信は一切行われません。

また、本ポリシーページ自体も Google Analytics の標準的なウェブサイト解析タグにより、訪問状況（匿名のアクセスログ）が記録されます。これはアプリ内の利用状況データとは別の、一般的なウェブサイト解析です。

4. 動作環境・通信について

本アプリは、同一PC上にインストールされた Audio Precision社の APx500 アプリケーションと、.NET（pythonnet）経由でローカル通信を行うことで測定器を操作します。測定器の操作・測定データに関する通信は、インターネットを含む外部ネットワークを経由しません。

第2項に記載の利用状況データ（匿名）のみ、Google Analyticsへ送信するため外部ネットワーク通信を行います。通信が失敗してもアプリの動作には影響しません。

Privacy Policy & Disclaimer

Last updated: June 24, 2026

1. Overview

APx Controller is a Windows desktop application developed by NARUKAMI AKUT for remotely controlling Audio Precision APx500-series analyzers (such as the APx515). This policy explains how the app handles information.

2. Information We Collect

Data TypePurposeTransmitted
Analyzer settings / measurement data Remote control and display of the APx500 application No (local PC only)
Report output folder path Specifying where sequence reports are saved No (local PC only)
Usage data (anonymous ID, launch time, app version, OS version, and exception type only) Understanding usage and improving app quality Yes (Google Analytics, aggregated and anonymous only)

The app does not require an account or login and does not collect any personally identifiable information such as name or email address. Settings and measurement data are held in memory only while the app is running and are never written to disk or sent to a server.

The anonymous ID (a randomly generated UUID) used for usage data is created automatically on first launch and stored in a local settings file on your PC. It does not identify you personally. Measurement values, analyzer settings, file paths, and usernames are never transmitted. The only information sent is whether the app was launched, which version is in use, and what type of error occurred (the error message text and source location are not included).

This transmission also uses the IP address of the connection, which Google Analytics converts into an approximate country/region (used to aggregate where the app is used). The IP address itself is not stored and is discarded after conversion.

3. Third-Party Services

To understand launch counts and improve app quality, the app sends anonymous usage data using Google Analytics (GA4 Measurement Protocol). The data sent is as described in Section 2 and is not used for advertising. The app does not use any dedicated crash-reporting service.

This transmission can be disabled at any time via the "Send usage data" checkbox on the main screen. When disabled, no data described in this section is sent.

This policy page itself also uses Google Analytics' standard website analytics tag to record anonymous visit data (page views). This is separate from the in-app usage data above and is standard website analytics.

4. System Requirements & Communication

The app controls the analyzer by communicating locally, via .NET (pythonnet), with the Audio Precision APx500 application installed on the same PC. Communication related to analyzer operation and measurement data does not occur over the internet or any external network.

Only the anonymous usage data described in Section 2 is sent over an external network, to Google Analytics. If this communication fails, it does not affect the app's operation.

Measurement values and settings displayed or controlled by this app depend on the connected APx500 application and analyzer hardware. The app itself does not guarantee the accuracy of any measurement values.
